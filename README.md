Summary: Автозамена изображений для Out-of-Stock позиций

Добавил в пайплайн парсера проверку доступности товара. Теперь, если позиция попадает под критерии «неактивна», вместо пустых или неактуальных фото прокидываем заглушку.
Логика работы:
Триггеры на замену:
Статус: Выведен из ассортимента, К выведению или Снято с производства.
Остаток на складах: 0 (или null).
Действие: Поля PREVIEW_PICTURE и DETAIL_PICTURE перетираются ссылкой на no-image-placeholder.

Зачем это нужно:
Чистим витрину от визуального мусора.
Унифицируем отображение архивных товаров для SEO-индекса (чтобы карточки не висели с битыми картинками).

Технические детали:
Проверку вынес в отдельный метод checkProductAvailability(), чтобы не загромождать основной цикл парсинга. Константы со статусами вывел в конфиг.
